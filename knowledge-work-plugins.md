# Knowledge Work Plugins

> Claude Cowork(Desktop) 전용 역할별 플러그인 패키지

## 소개
Anthropic이 제공하는 Claude Cowork(Desktop) 전용 공식 플러그인이다. 세일즈, 재무, 법무, 마케팅, HR, 엔지니어링 등 역할별로 특화된 플러그인을 포함하며, 팀 전체에 배포 가능한 organization-managed plugin 형태로 제공된다. Cowork paid plan 또는 Claude Desktop 전용이며 CLI 환경에서는 사용 불가.

## 주요 기능
- Sales: 세일즈 자동화
- Finance: 재무 분석
- Legal: 법무 검토
- Marketing: 마케팅 캠페인
- HR: 인사 관리
- Engineering Design: 엔지니어링 설계
- Operations: 운영 관리
- Data Analysis: 데이터 분석
- Plugin Creation: 플러그인(Plugin) 개발

## 공식 링크
- GitHub: https://github.com/anthropics/knowledge-work-plugins

## 설치
사용 중인 플랫폼(Claude Code, Codex, Cursor 등)의 플러그인/스킬 설치 방식에 따라 공식 저장소를 참고하여 설치.
