# Obsidian Skills

> Obsidian 볼트(Vault)를 AI 에이전트가 자동화할 수 있도록 가르치는 스킬 모음

## 소개

Obsidian Skills는 Obsidian CEO(kepano)가 직접 제작한 스킬 모음이다.
Markdown, Bases, JSON Canvas 사용법과 CLI 활용법을 에이전트에게 가르쳐 볼트 자동화를 지원한다.
플랫폼에 독립적인 구조로, Obsidian을 사용하는 모든 환경에서 활용 가능하다.

## 주요 기능

- obsidian-markdown: Obsidian Flavored Markdown (wikilink, callout, properties)
- obsidian-bases: Bases 파일 (뷰, 필터, 수식, 요약)
- json-canvas: JSON Canvas (노드, 에지, 그룹)
- obsidian-cli: CLI 기반 볼트 상호작용 + 플러그인/테마 개발
- defuddle: 웹 페이지를 깨끗한 마크다운으로 추출 (토큰 절약)
