# 프로젝트 계약서

## 명령어
- 설치: `<패키지 매니저> install`
- 개발: `<패키지 매니저> dev`
- 테스트: `<패키지 매니저> test`
- 린트: `<패키지 매니저> lint`
- 빌드: `<패키지 매니저> build`

## 경계
- API: `src/api/`
- 도메인 로직: `src/domain/`
- UI: `src/components/`

## 안전 규칙
- `.env*` 파일 절대 수정 금지
- 파일 삭제 전 반드시 확인
- 커밋 전 테스트 실행
