## 금지 사항 (반드시 대안 포함)
- `.env*` 파일 수정 금지 → 환경변수는 `.env.example`에 키만 기록
- `rm -rf` 사용 금지 → 삭제 전 사용자 확인
- 테스트 없이 커밋 금지 → 최소 관련 테스트 1개 실행
- 직접 프로덕션 배포 금지 → PR을 통해서만
